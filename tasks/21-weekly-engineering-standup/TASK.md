Conduct weekly engineering standup with Backend Engineer, Frontend Engineer, and DevOps Engineer.

## Process

1. Review completed work from past week
2. Review in-progress work and blockers
3. Coordinate cross-service dependencies (API contracts, deployment needs)
4. Review code quality and technical debt
5. Plan next week's priorities aligned with MVP launch goal
