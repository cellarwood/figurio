Review intellectual property compliance for all figurine designs.

## Checklist
1. Verify all new catalog models have valid commercial licenses or are original commissions
2. Review AI-generated custom figurines for resemblance to copyrighted characters
3. Check content moderation logs for flagged prompts
4. Review any DMCA or takedown requests received
5. Update the blocked-prompts list based on new IP concerns
6. Consult with legal if any edge cases arise

## Output
- IP compliance report
- Updated blocked-prompts list
- Legal consultation notes if needed
