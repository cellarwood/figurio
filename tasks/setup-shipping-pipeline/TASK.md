Configure shipping providers and define the fulfillment pipeline for delivering figurines to customers.

## Requirements

### Shipping Providers
- **Czech Republic:** Zásilkovna (Packeta) — pickup points + home delivery
- **EU:** DHL standard parcel
- Get accounts set up, API credentials, and rate sheets for both

### Packaging
- Define box sizes for each figurine tier (Small, Medium, Large)
- Protective packaging: foam insert or bubble wrap to prevent damage
- Branded elements: logo on box, thank-you card, care instructions
- Get quotes from packaging suppliers

### Fulfillment SOP
- Document the complete flow: order paid → print file to MCAE → figurine received from MCAE → quality check → package → ship → tracking number to customer
- Define who does what at each step
- Handling for damaged-in-transit: replacement policy, insurance

## Deliverables

- Shipping provider accounts configured
- Rate sheet (shipping cost by destination zone and package size)
- Packaging specification document
- Fulfillment SOP document
