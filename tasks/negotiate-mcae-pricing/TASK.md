This is the #1 operational priority. Customer-facing prices cannot be set until MCAE per-unit costs are locked in.

## Negotiation Objectives

### Per-Unit Pricing
- Get firm quotes for all three size tiers:
  - Small (~8cm height)
  - Medium (~15cm height)
  - Large (~25cm height)
- Request pricing at volume tiers: 1-10 units, 10-50 units, 50-100 units, 100+ units
- Clarify: does pricing vary by model complexity (polygon count, color count)?
- Request separate pricing for standard PolyJet vs. any lower-cost alternatives they offer

### SLA Terms
- Production turnaround time per size tier (calendar days from file submission to print completion)
- Maximum defect rate and replacement policy
- Communication protocol: how to submit files, how to track production status
- Minimum order requirements (if any)
- Capacity: maximum monthly throughput they can commit to

### Logistics
- Shipping options: MCAE ships to Figurio warehouse vs. direct-to-customer drop-ship
- Packaging: does MCAE provide any packaging, or do we need to supply branded packaging?
- File format requirements: what format do they need for print files?

## Deliverables

- Pricing sheet: per-unit cost across all size tiers and volume levels
- SLA document: turnaround times, defect policy, capacity commitments
- COGS model: MCAE cost + packaging + shipping = total cost per unit per size per region
- Margin analysis: recommended customer-facing prices at target margins (50-70%)

## Acceptance Criteria

- Firm pricing quotes received from MCAE for all size tiers
- SLA terms documented and agreed upon
- COGS model shared with CEO for pricing decisions
