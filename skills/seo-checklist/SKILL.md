# SEO Checklist

## On-Page SEO (every page/post)

- [ ] **Title tag:** Under 60 characters, includes primary keyword, compelling
- [ ] **Meta description:** Under 160 characters, includes keyword, has CTA
- [ ] **H1:** One per page, includes primary keyword
- [ ] **URL slug:** Short, descriptive, kebab-case (e.g., `/blog/how-ai-creates-custom-figurines`)
- [ ] **Header hierarchy:** H1 → H2 → H3, no skipped levels
- [ ] **Primary keyword:** In title, H1, first paragraph, and 2-3 times naturally in body
- [ ] **Internal links:** At least 2 links to other pages (product pages, blog posts, category pages)
- [ ] **Image alt text:** Descriptive, includes keyword where natural
- [ ] **Image optimization:** WebP format, compressed, lazy-loaded

## Technical SEO

- [ ] Sitemap.xml generated and submitted to Google Search Console
- [ ] robots.txt configured (allow indexing of public pages)
- [ ] Canonical URLs set on all pages
- [ ] Open Graph tags for social sharing (title, description, image)
- [ ] Schema markup: Product (for catalog items), Article (for blog posts), Organization
- [ ] Page load speed under 3 seconds (Core Web Vitals)
- [ ] Mobile-friendly (responsive design)
