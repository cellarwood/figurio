# SEO Checklist

## On-Page Checklist (every page/post)

- [ ] **Title tag:** 50-70 chars, primary keyword near the start
- [ ] **Meta description:** 150-160 chars, includes keyword and CTA
- [ ] **H1:** Exactly one per page, matches title intent
- [ ] **H2/H3:** Logical hierarchy, include secondary keywords naturally
- [ ] **URL slug:** Short, kebab-case, includes primary keyword (`/blog/custom-3d-printed-figurines`)
- [ ] **Images:** Descriptive alt text with keywords, compressed (WebP preferred, < 200KB)
- [ ] **Internal links:** 2-3 links to other pages/posts per article
- [ ] **External links:** 1-2 authoritative sources if relevant (e.g., linking to 3D printing explainers)
- [ ] **Word count:** 800+ for blog posts (aim for 1,200 for competitive keywords)
- [ ] **Mobile rendering:** Verify content renders correctly on 375px viewport

## Technical SEO

- Sitemap.xml generated and submitted to Google Search Console
- Robots.txt allows crawling of all public pages
- Canonical URLs on all pages
- Schema.org Product markup on product pages (name, price, image, availability)
- Schema.org Article markup on blog posts
- Page load speed < 3 seconds (Core Web Vitals)
- No broken links (check monthly)

## Content Calendar SEO Integration

When planning content:
1. Pick a primary keyword from the cluster list
2. Check search volume and competition (use free tools: Google Keyword Planner, Ubersuggest)
3. Write content that directly answers the search intent
4. Include 3-5 secondary keywords naturally
5. Plan internal linking structure before publishing
