# Figurio SEO Checklist

Use this skill when creating, optimizing, or reviewing any page that should capture organic demand for Figurio.

## SEO Goal

Rank for the searches people actually use when they are looking for custom figurines, giftable collectibles, seasonal buying ideas, and related product intent. The page should satisfy search intent first and convert second.

## SEO Principles

- Match the page to one clear search intent.
- Use the same language customers use when they search.
- Make the page useful without turning it into keyword padding.
- Align headings, metadata, copy, and internal links around the same topic.
- Write for humans who may buy, not for search engines alone.

## Intent Types

- Transactional: ready to buy or compare products.
- Commercial investigation: trying to choose a product or use case.
- Informational: learning about a custom figurine, gifting idea, or process.
- Seasonal: looking for holiday, event, or occasion-based content.

If the intent is mixed, make the primary intent obvious and support it with the secondary intent only where it helps.

## Page Build Checklist

### 1. Keyword and Topic Mapping

- Pick one primary keyword or topic.
- Add a small set of secondary variants that are genuinely relevant.
- Match the keyword to the page type and funnel stage.
- Avoid creating multiple pages that compete for the same intent.

### 2. Title And Meta

- Put the primary topic early in the title.
- Make the meta description specific, readable, and clickable.
- Reflect the page's real offer, not a generic teaser.
- Keep titles and descriptions distinct across similar pages.

### 3. Headings And Structure

- Use a single clear H1 that reflects the page intent.
- Build H2s around user questions, benefits, process, proof, and next steps.
- Keep sections scannable and logically ordered.
- Do not bury the core topic under long brand storytelling.

### 4. Body Copy

- Answer the searcher's main question early.
- Include supporting details that matter for purchase confidence.
- Use natural keyword variants where they belong.
- Keep the copy specific to Figurio's products and process.

### 5. Internal Linking

- Link to closely related product, collection, or support pages.
- Use descriptive anchor text.
- Point users toward the next logical step in the journey.
- Avoid orphan pages that cannot be discovered from relevant hubs.

### 6. Media And Alt Text

- Use images that support the exact topic of the page.
- Write alt text that describes the image honestly and concisely.
- Do not stuff keywords into alt text.
- Make sure visuals reinforce the page's intent and conversion goal.

### 7. Schema And Technical Signals

- Add structured data where it matches the page type.
- Keep canonical tags aligned with the preferred URL.
- Confirm indexability, crawlability, and correct metadata output.
- Ensure mobile rendering does not hide essential content.

## Figurio-Specific SEO Rules

- Use product language that fits the actual catalog and customization model.
- Include gifting and occasion language where it naturally fits the page.
- Reflect production timing, personalization, and approval steps when they affect intent.
- Build seasonal pages early enough to earn visibility before peak demand.
- Reuse canonical naming for product lines, custom options, and collections.

## Content Strategy Rules

- Create pages that answer distinct search jobs instead of duplicating the same topic.
- Support commercial pages with helpful explanatory content where needed.
- Use seasonal campaigns to target timely queries without compromising evergreen relevance.
- When a page needs both SEO and conversion, keep the search intent visible while preserving the CTA.

## Quality Checks Before Publish

- Does the page have one obvious primary keyword or topic?
- Does the title clearly match the page intent?
- Is the content useful enough to rank and specific enough to convert?
- Are internal links helping the user move deeper into the site?
- Is the page distinct from other Figurio pages covering adjacent topics?
- Are metadata, headings, and body copy aligned?
- Does the page avoid keyword stuffing and vague filler?

## Common Failure Modes

- Multiple pages targeting the same search intent.
- Generic copy that could belong to any ecommerce brand.
- Titles that are clever but not searchable.
- Thin pages that only restate the headline.
- Seasonal pages launched too late to matter.
- Internal links that point nowhere useful for the customer journey.
