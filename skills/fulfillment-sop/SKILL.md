# Fulfillment SOP

## Order Lifecycle

```
Customer pays → Order created → Print file prepared → Sent to MCAE →
MCAE prints → QA check → Packaging → Ship to customer → Delivered
```

## Step-by-Step Procedure

### 1. Order Received
- **Trigger:** Stripe webhook `checkout.session.completed` or `payment_intent.succeeded`
- **Action:** Order record created with status `paid`
- **Verify:** Payment amount matches order total, customer details complete

### 2. Print File Preparation
- **Catalog items:** Print file already validated and stored. Retrieve from product record.
- **AI-custom items:** Print file generated by AI pipeline, validated by mesh repair. Retrieve from `ai_jobs` record.
- **Action:** Bundle print files with order metadata (size tier, quantity, special instructions)
- **Status update:** `preparing`

### 3. Submit to MCAE
- **Action:** Send print files to MCAE via agreed protocol (email, FTP, API — per SLA)
- **Include:** Order ID, size tier, quantity, delivery deadline
- **Status update:** `printing`
- **Track:** Note MCAE confirmation/job number

### 4. Production Tracking
- **Check:** MCAE production status per SLA turnaround times
- **Escalate:** If past SLA deadline, contact MCAE immediately
- **Defect handling:** If MCAE reports a print failure, request reprint. Log defect.

### 5. Quality Assurance
- **If Figurio receives the print:** Visual inspection before shipping
- **If MCAE ships direct:** Rely on MCAE QA + spot-check via customer feedback
- **Defect threshold:** >3% defect rate triggers vendor review meeting

### 6. Packaging
- **Branded box:** Figurio-branded packaging with protective insert
- **Include:** Thank-you card, care instructions, social media share CTA (discount code for posting)
- **Size-appropriate:** Small box for 8cm, medium for 15cm, large for 25cm

### 7. Shipping
- **Czech Republic:** Zásilkovna — pickup point or home delivery
- **EU:** DHL standard parcel
- **Action:** Generate shipping label, attach tracking number to order
- **Status update:** `shipped`
- **Notify customer:** Email with tracking number and estimated delivery

### 8. Delivery Confirmation
- **Trigger:** Carrier tracking shows delivered
- **Status update:** `delivered`
- **Action:** Send follow-up email (review request, share CTA)

## Turnaround Targets

| Product Type | Total Customer Wait |
|-------------|-------------------|
| Catalog (pre-validated) | 7-10 business days |
| AI-Custom | 10-15 business days |

## Exception Handling

| Issue | Action |
|-------|--------|
| Defective print | Reprint at MCAE's cost per SLA. Ship replacement. Notify customer with apology + timeline. |
| Lost in shipping | File claim with carrier. Ship replacement. Refund shipping if >5 day delay. |
| Customer complaint | Respond within 24 hours. Offer replacement or refund per case severity. |
