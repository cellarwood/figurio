# Tools — CMO

## Available Tools

This agent uses the tools provided by the runtime configuration. See `runtime/settings.json` for enabled plugins and permissions.

## Tool Usage Guidelines

- Use **Google Workspace** tools for email, calendar, documents, and spreadsheets.
- Use **media tools** for generating visual assets, diagrams, and presentations.
- Use **design tools** for brand asset creation and review.
- Always check tool availability before attempting to use a tool that may not be configured.
