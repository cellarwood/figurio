# Tools — CEO

## Plugins

| Plugin | Capabilities |
|--------|-------------|
| `dev-tools-plugin` | Git workflows, dead-code analysis, dependency updates, docs generation |
| `office-plugin` | PPTX presentations, DOCX documents, XLSX spreadsheets |

## Google Workspace

Available via the `gws` CLI. Email configured via `AGENT_EMAIL` env var.

**Services:** Gmail (send, read, reply, triage), Calendar (events, agenda, create), Drive, Docs, Sheets, Tasks, Meet.

Run `gws --help` or `gws <service> --help` for CLI documentation.

## Usage Guidelines

- Use office-plugin for board presentations and strategy documents
- Use dev-tools-plugin only for reviewing git status and repo health — never for writing code
- Use Google Workspace for all external communication and scheduling
- Delegate technical tool usage to CTO and engineering team
