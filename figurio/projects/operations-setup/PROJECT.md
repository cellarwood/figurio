# Operations Setup

Establish the complete production and fulfillment pipeline: MCAE pricing, shipping integrations, fulfillment workflow, packaging, and quality control.

## Scope

- Negotiate per-unit pricing with MCAE across all size tiers
- Integrate Zasilkovna API for Czech Republic shipping
- Integrate DHL API for international shipping
- Design fulfillment workflow (order -> MCAE -> ship -> deliver)
- Design and source branded packaging
- Establish quality control procedures

## Deliverables

1. MCAE pricing agreement (per-unit cost for Small, Medium, Large)
2. Shipping API integrations (label generation, tracking)
3. Fulfillment workflow documentation and automation
4. Branded packaging design and supplier
5. QC checklist and defect handling process
