# Engineering

Owns all technology at Figurio: the web storefront, backend API, AI/ML pipeline, infrastructure, and CI/CD. Responsible for building, testing, deploying, and maintaining every system the company runs on.

## Responsibilities

- Web storefront (React/TypeScript) -- catalog, cart, checkout, order tracking, AI prompt UI
- Backend API (Python/FastAPI) -- product catalog, orders, payments, shipping, authentication
- AI/ML pipeline -- text-to-3D generation, mesh repair, printability validation, preview rendering
- Infrastructure -- Docker, Kubernetes, Helm, Traefik, CI/CD, monitoring
- Database -- PostgreSQL schema, migrations, performance
- Integrations -- Stripe, Zasilkovna, DHL, text-to-3D APIs
