## Scope

Build the customer-facing web application for Figurio. Includes product catalog browsing with filters and search, shopping cart, Stripe checkout integration, AI custom figurine prompt interface with 3D preview, user account management, and SEO-optimized marketing pages.

## Tech Stack

- TypeScript with strict mode
- React with shadcn-ui and Radix UI
- Tailwind CSS for styling
- GSAP for animations

## Success Criteria

- Responsive design across mobile, tablet, desktop
- WCAG 2.1 AA accessibility compliance
- Sub-3s page load on 4G connection
- Smooth checkout conversion flow
