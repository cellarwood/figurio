# Infrastructure

## Scope

- Monorepo structure (apps/web, services/api, packages/shared, infra/)
- Docker multi-stage builds for frontend and backend
- Kubernetes deployment on microk8s-local
- CI/CD with GitHub Actions
- Traefik ingress with Let's Encrypt SSL
- Monitoring dashboards

## Success Criteria

- Automated deploy pipeline
- Zero-downtime deployments
- Monitoring dashboards for all services
