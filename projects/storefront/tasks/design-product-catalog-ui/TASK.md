Build the product catalog page for the Figurio storefront.

- Grid layout displaying figurine cards with thumbnail, name, price, and size options
- Category filtering (trending, seasonal, evergreen, custom)
- Text search with instant results
- Sort options (newest, popular, price low-high, price high-low)
- Pagination or infinite scroll for large catalogs
- Responsive layout: 1 column on mobile, 2-3 on tablet, 4 on desktop
- Use shadcn-ui components and Tailwind CSS
- Fetch product data from the backend catalog API
