# Build 3D model preview viewer component

Build interactive 3D model viewer using Three.js/React Three Fiber. Features:

- Orbit controls
- Zoom
- Lighting
- Progressive loading for large meshes
- Fallback to static rendered images if WebGL unavailable

Used on product detail pages and AI custom preview.
