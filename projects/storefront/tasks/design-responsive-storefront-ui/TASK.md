Design and implement the overall storefront layout and navigation.

- Mobile-first responsive layout using Tailwind CSS breakpoints
- Header with logo, navigation (catalog, custom, about), cart icon, user menu
- Footer with company info, links, social media icons
- Homepage hero section showcasing featured figurines and the "Prompt to Print" feature
- Navigation: hamburger menu on mobile, horizontal nav on desktop
- Cart sidebar/modal with item summary and checkout button
- User account pages: login, register, profile, order history, order detail with tracking
- Loading states, error states, and empty states for all pages
- GSAP animations for page transitions and hero section
- Consistent use of shadcn-ui components throughout
