## Scope

Build and deploy the Figurio e-commerce storefront with these capabilities:

- Product catalog with filtering, search, and category browsing
- Product detail pages with interactive 3D model viewer and size selection
- Stripe checkout supporting cards, Apple Pay, Google Pay, SEPA, iDEAL, and Bancontact
- User accounts with registration, login, order history, and order tracking
- Responsive, mobile-first design optimized for social media traffic
- Order management API for the full lifecycle (create → track → fulfill → deliver)

## Success Criteria

- Storefront deployed to production at figurio.cellarwood.org
- End-to-end purchase flow working (browse → select → pay → track)
- 3D model viewer renders catalog figurines smoothly on mobile and desktop
- Stripe processes test payments successfully across all supported methods
- User can create account, place order, and view order status
