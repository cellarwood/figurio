## Scope

Build the customer-facing storefront for Figurio.

### Key deliverables

- Catalog browsing with filtering, search, and product detail pages
- AI figurine prompt interface with real-time 3D preview
- Shopping cart with size tier selection
- Stripe checkout flow (credit/debit, Apple Pay, Google Pay)
- Responsive design for mobile and desktop
- shadcn-ui component library with Tailwind styling

## Success Criteria

- Lighthouse score > 90 for performance and accessibility
- Complete purchase flow tested end-to-end
- 3D preview renders correctly on major browsers
- Mobile-first responsive design verified
