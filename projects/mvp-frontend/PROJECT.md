## Scope

Build the Figurio customer-facing storefront as a React/TypeScript single-page application. Covers product catalog with filtering, 3D model preview, AI-prompted custom figurine interface, shopping cart, Stripe checkout, and order tracking.

## Tech Stack

- React with TypeScript (strict mode)
- shadcn-ui + Radix UI for components
- Tailwind CSS for styling
- GSAP for animations
- npm for package management

## Success Criteria

- Catalog browsing with category/size filters
- 3D model preview on product pages
- AI prompt flow with preview approval
- Stripe checkout with Apple/Google Pay
- Order tracking dashboard
- Responsive design (mobile-first)
