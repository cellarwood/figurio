## Scope

Build the customer-facing storefront and admin dashboard for Figurio using React/TypeScript.

## Key Deliverables

- Product catalog pages with filtering and search
- Shopping cart and Stripe Elements checkout flow
- User authentication pages
- Order history and status tracking
- Admin dashboard for catalog and order management
- Responsive design for mobile and desktop

## Tech Stack

- React with TypeScript (strict mode)
- shadcn-ui + Radix UI primitives
- Tailwind CSS for styling
- Vite for build tooling
- npm for package management

## Success Criteria

- All pages responsive (mobile-first)
- Checkout flow tested end-to-end with Stripe
- WCAG 2.1 AA accessibility compliance
- Lighthouse performance score > 90
