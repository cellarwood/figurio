## Scope

Build the core backend API for the Figurio e-commerce platform.

### Key deliverables

- Product catalog API (CRUD, search, filtering by category/size/price)
- Order management (lifecycle, status tracking, customer notifications)
- Stripe payment integration (checkout sessions, webhooks, refunds)
- AI figurine generation pipeline (text-to-3D API integration, job management, preview delivery)
- PostgreSQL database with proper schema design

## Success Criteria

- All API endpoints documented and tested
- Stripe webhooks handling all payment events correctly
- AI generation pipeline producing printable 3D models from text prompts
- 95%+ test coverage on payment and order logic
