# Build Storefront Frontend

Implement the React/TypeScript web storefront with product browsing, 3D viewer, cart, and checkout.

## Expected Subtasks

- CTO delegates to Frontend Engineer
- Homepage with featured/trending figurines
- Catalog page with grid layout, filters, search
- Product detail page with interactive 3D model viewer (React Three Fiber), size selector, add-to-cart
- Shopping cart with quantity management and size tier pricing
- Checkout page integrated with Stripe Elements
- Customer account pages (registration, login, order history, order detail)
- Order tracking page with status timeline
- Responsive design (mobile-first), accessibility compliance
- Navigation, footer, 404, loading states
