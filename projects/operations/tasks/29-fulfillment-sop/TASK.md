Fulfillment SOP:
- Order received → print file prepared → sent to MCAE
- MCAE prints → ships to Figurio (or drop-ships)
- Quality inspection checklist
- Packaging procedure per size tier
- Shipping label generation and carrier handoff
- Customer notification triggers
- Returns and refund handling process
- Exception handling (damaged prints, delays)
