# Evaluate Text-to-3D Services

Benchmark candidate text-to-3D APIs to select the best provider for Figurio's custom figurine pipeline.

## Expected Subtasks

- CTO delegates to ML Engineer
- Evaluate: Meshy, Tripo3D, Luma Genie, CSM 3D (and any newer entrants)
- Benchmark criteria: model quality (detail, color accuracy), printability (manifold geometry out of the box), generation speed, API cost per model, reliability/uptime
- Generate 20+ test models across different prompt styles (characters, animals, abstract, detailed scenes)
- Score each service and produce a recommendation report
- Assess self-hosted model option for long-term cost savings
