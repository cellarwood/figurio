Create the foundational brand identity for Figurio that guides all visual and verbal communication.

## Deliverables

### Brand Style Guide Document

1. **Logo concept** — describe the logo direction (wordmark, icon, or combination), visual metaphor, and mood. Provide detailed creative brief for a designer/AI tool to execute.

2. **Color palette**
   - Primary: 2-3 brand colors
   - Secondary: 2-3 supporting colors
   - Neutral: grays for text and backgrounds
   - Include hex codes and usage guidelines

3. **Typography**
   - Heading font (display/personality)
   - Body font (readability)
   - Source: Google Fonts or open-source for web use

4. **Brand voice and tone**
   - Personality traits (e.g., playful, creative, warm, slightly quirky)
   - Dos and don'ts for writing style
   - Examples of on-brand vs off-brand copy
   - Tone adjustments by context (social media vs email vs product page)

5. **Visual direction**
   - Photography style for product shots
   - Background and staging guidelines
   - Social media visual template direction

## Constraints

- All visual assets must be IP-clean (no copyrighted characters)
- Colors must meet WCAG AA contrast requirements for web use
- Fonts must be free for commercial web use

## Acceptance Criteria

- Brand style guide document completed
- Shared with Frontend Engineer for implementation
- Shared with Content Creator for content alignment
