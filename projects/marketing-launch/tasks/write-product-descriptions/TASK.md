Write SEO-optimized product descriptions for all initial catalog figurines.

- Write unique, evocative descriptions for each catalog figurine (150-300 words each)
- Lead with what makes each figurine special (pose, detail, story) not dimensions
- Include SEO keywords naturally: "3D printed figurine", "custom collectible", "full color figurine"
- Write size tier descriptions (Small ~8cm, Medium ~15cm, Large ~25cm) with use case suggestions
- Write category descriptions for catalog sections (fantasy, seasonal, pop culture, animals)
- Include care instructions and material information for product pages
- Ensure zero copyrighted character references in all descriptions
