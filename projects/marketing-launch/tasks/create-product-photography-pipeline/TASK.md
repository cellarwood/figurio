Establish a repeatable process for photographing figurines for the catalog and marketing.

- Photography setup guide: recommended lighting (softbox or ring light), background (white sweep, lifestyle settings), camera/phone settings
- Shot list per figurine:
  - Hero shot (front, centered, clean background)
  - 3/4 angle shot
  - Detail close-up (face, texture, colors)
  - Lifestyle shot (figurine on desk, shelf, next to objects for scale)
  - Size comparison shot (figurine next to common object like a coffee mug)
- Image specs: resolution, aspect ratios for web (16:9 hero, 1:1 catalog, 4:5 Instagram)
- Naming convention: {figurine-slug}_{shot-type}_{size}.{ext}
- Before physical figurines are available: define how to create marketing-quality 3D renders as stand-ins
- Post-processing guidelines: color correction, background removal, consistent style
